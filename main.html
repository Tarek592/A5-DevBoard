Question-1: What are the different ways to select an element in the DOM?

There are many different ways to select an element in the Dom. Example:-

  • getElementById()-Selects an element by its id name.
  • getElementsByClassName()-elects elements by their class name.
  • getElementsByTagName()-Selects elements by their tag name .
  • querySelector()-Selects the first element that matches a class id or tag names.
  • querySelectorAll()-Selects all elements that match a specified class ,id or tag selector.
  • getAttribute()-Allows select elements based on their attributes.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The difference between innerHtml, innerText and textContent is given by itself

innerHTML:- It returns or sets the HTML markup inside the element, including tags.

innerText:- It returns or sets the "manually" text content of an element. This means it only gets the text visible to the user, ignoring any HTML tags.

textContent:- It returns or sets the raw text content, also ignoring HTML tags.

Question-3: What is event delegation in the DOM?

Event Delegation is a pattern for handling DOM events in JavaScript. It adds the event handler to the parent element and the child element's event handler to the event element.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process where an event starts from its innermost target element and progressively reaches its parent elements.

Question-5: How do you create, add, and remove elements using JavaScript?

JavaScript can be used to manipulate the DOM (Document Object Model), whereby new HTML elements can be created, or add and removed.

createElement is a JavaScript method used to create new HTML elements in the Document Object Model (DOM).

appendChild() is a JavaScript method used to append a new child element to the end of an existing parent element.

removeChild() method is used to remove an element from the DOM. It removes a child element from a parent element.
